# План доработки: ревью паритета v2

**Дата:** 2026-04-01  
**Статус:** В работе  
**Приоритет:** P1

## Проблема

Ревью выявило 3 ключевых разрыва между новым и оригинальным сайтом:

1. **[P1] Главная урезана** — потеряны блоки «Наши преимущества» (6 карточек), «Наш подход к обучению» (описание + статистика 3 показателя), «Наши программы» (3 института с описаниями). Текущая главная: Hero → News → Institutes → Promotions → Newsletter.

2. **[P1] Legacy HTML без CSS** — на контентных страницах body_html содержит ~30 уникальных CSS-module префиксов (typography_*, articles-form_*, collapsible_*, schedule-prices_* и т.д.) без соответствующих стилей. Результат: «плоский» контент без структуры.

3. **[P2] Две визуальные системы** — главная в новом дизайне, контентные страницы — сырой legacy HTML. Визуально выглядит как два разных сайта.

## Стратегия

**Компонентный rebuild** (не legacy-CSS-прослойка), потому что:
- Legacy классы — CSS Modules с хешами, нет смысла воспроизводить
- Структурированные данные уже в entities JSON, можно рендерить компонентами
- Поддерживаемость: свои компоненты с Astro scoped styles >> raw HTML с мёртвыми классами

## Данные с оригинального сайта (извлечены для реализации)

### Преимущества (6 карточек)

1. **Наш многолетний опыт в образовании**  
   Подготовка квалифицированных специалистов в медицине — сложная и ответственная задача. У нас получается делать это лучше других, потому что наработали годами проверенные методики.

2. **Сотрудничество с зарубежными институтами**  
   ИКПК — единственный официальный представитель институтов Апледжера и Барраля в России, предлагающий уникальные методики ведущих специалистов.

3. **Настоящие преподаватели – ключ к успеху**  
   В педагогическом штате ИКПК — лучшие из лучших, создавшие авторские программы и щедро делящиеся своими знаниями.

4. **Документы об образовании для наших студентов**  
   Студенты с базовым медицинским образованием получают удостоверения о повышении квалификации с внесением сведений в ФИС ФРДО + ЗЕТ/баллы НМО.

5. **Издательство медицинской литературы**  
   Издательский дом ИКПК занимается лицензированным переводом и изданием на русском языке научной и учебной литературы по остеопатии и прикладной кинезиологии.

6. **Практика в нашем медицинском центре**  
   В медицинском центре студенты могут под руководством опытных преподавателей совершенствовать знания — аналог интернатуры в области холистической медицины.
